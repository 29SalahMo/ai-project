# 🚀 Deploy Your App Now - Quick Steps

## Step 1: Deploy to Streamlit Cloud (2 minutes)

1. **Go to:** https://share.streamlit.io
2. **Sign in** with your GitHub account (29SalahMo)
3. Click **"New app"** button
4. Fill in:
   - **Repository:** `29SalahMo/ai-project`
   - **Branch:** `main`
   - **Main file path:** `app.py`
5. Click **"Deploy"**

## Step 2: Get Your App URL

After deployment (takes 1-2 minutes), you'll get a URL like:
```
https://ai-project-29salahmo.streamlit.app
```

## Step 3: Update README with Your Actual URL

Once you have your Streamlit Cloud URL, update the README.md file:

1. Open `README.md`
2. Find the line: `https://ai-project-29salahmo.streamlit.app`
3. Replace it with your actual URL from Streamlit Cloud
4. Commit and push:
   ```bash
   git add README.md
   git commit -m "Update live demo URL"
   git push
   ```

## Step 4: Add Link to GitHub Repository Description

1. Go to: https://github.com/29SalahMo/ai-project
2. Click the **⚙️ Settings** icon (gear icon) next to "About"
3. Check **"Website"** and paste your Streamlit Cloud URL
4. Click **"Save changes"**

**Done!** Now users will see the link on your GitHub repository! 🎉
